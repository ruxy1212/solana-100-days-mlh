
I've been building on Solana lately, creating wallets on Devnet, funding them from faucets, transferring assets between wallets, and experimenting with Token-2022.

So far, I've minted tokens with extensions like Freeze Authority, Interest-Bearing Tokens, and a few others. One thing that really stood out to me is how these extensions can be combined to enforce real-world, off-chain compliance requirements, with no custom smart contracts required.

As a practical example, I built a Revocable Credential by combining the Non-Transferable and Permanent Delegate Token Extensions, directly at the token level. 

I wrote a deep dive on how it works: https://dev.to/russell_oje/soulbound-credentials-on-solana-b5c
