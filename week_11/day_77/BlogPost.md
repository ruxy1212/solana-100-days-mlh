Last week I built a program using Cross-Program Invocation on Solana!

A Cross-Program Invocation (CPI) is just like a function call with a guest list. Where your program is the host. The program you are calling is the venue. So the guest list is the set of accounts that venue needs to do its job:
- A program ID;
- The accounts it needs; and
- A signer authority.

This same single pattern called the System Program, Token-2022, and my even own counter program without changing shape.
