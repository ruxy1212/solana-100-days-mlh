# 100 Days of Solana

100 days of Solana challenge on MLH.

## Event Description

You’ve explored web3 but you’re not sure what to build or where to start.

100 Days of Solana is a hands-on challenge series that takes you from curiosity to capability.

Through daily, practical tasks, you’ll learn what you can build with Solana, why you’d choose it, and how to apply the skills you already have as a developer.

Each day is a small, concrete step. Together, they add up to real understanding.

By the end, you won’t just understand Solana, you’ll be ready to build something useful with it.

## The Roadmap

The 100 Days are divided into four major themes, which we call **Epochs**. Each Epoch consists of 4 **Arcs**. An Arc is a week of focused learning and building.

### The Weekly Rhythm of an Arc

Every Arc follows a structured cycle to ensure deep understanding:

- **Monday** ▶️ Try: Understand the core concept.
- **Tuesday** 🛠️ Build: Get the basic implementation working.
- **Wednesday** 📈 Stretch: Push the boundaries and add features.
- **Thursday** 📌 Reinforce: Review and solidify the logic.
- **Friday** 🧪 Experiment: Try your own creative approach.
- **Saturday** 📝 Document: Share your findings on DEV.to.
- **Sunday** 🚀 Amplify: Engage with the community.

## How to Run Locally

1. Install dependencies:
   - `pnpm install`
2. Add `.env`:
   - `KEY=your_key_here`
3. Start:
   - `node path-to-file.ext`
