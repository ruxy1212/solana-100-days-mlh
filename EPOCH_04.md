# Epoch 4: Shipping and Exploring

![Epoch 4 Hero](https://mlhusercontent.com/backgrounds/challenges/019f5b90-09c3-9a0f-9241-65ee784cc401.jpg?t=1783947594)

Welcome to Epoch 4. This stage is all about shipping your applications to production and exploring integration. We transition from sandboxed testing environments to deploying directly onto Solana mainnet-beta, configuring robust upgrade authorities, generating client-side SDKs with Codama, and developing responsive React frontends that interface cleanly with real-world browser wallets.
